# Informatiebehoefte — Noordje op weg naar thuis

> Uitwerking van de informatiebehoefte per subdoel, gebaseerd op de projectbrief
> "Afbouw JeugdzorgPlus en opbouw alternatieven 2024–2028".

## Hoofddoel

Jeugdigen **niet meer gesloten plaatsen**, niet meer buiten de provinciegrens,
maar **zo dicht mogelijk bij thuis en bij school**.

## Klantperspectief

Voor de jongere betekent dit: "Ik hoef niet ver van huis, niet in een gesloten
instelling, en de mensen die over mij beslissen hebben actuele informatie zodat
ik snel op de juiste plek terecht kan."

## Prioritering en Fasering

```
Fase 1 (Pilot Haarlemmermeer):
  → Subdoel 1a: Overzicht plaatsingen — basisset
  → Databronnen: iJw berichtenverkeer + aanbiederdata Haarlemmermeer

Fase 2 (Noord-Holland):
  → Subdoel 1a: Volledig plaatsingsoverzicht
  → Subdoel 1b: Voorzieningen overzicht
  → Databronnen: + CORV + CBS microdata

Fase 3 (Verdieping):
  → Subdoel 2: Voorspellende factoren (met DPIA)
  → Subdoel 3: Effectiviteit
  → Databronnen: + DUO + J&V/DJI

Fase 4 (Landelijk):
  → Alle subdoelen
  → Herbruikbaar voor andere bovenregionale samenwerkingsverbanden
```
