# Solution Documents — Compound Learning Index

> **Versie:** 1.0
> **Laatst bijgewerkt:** 2026-03-04

Dit is de centrale index van alle solution documents die zijn aangemaakt via het `/compound` command. Elke entry beschrijft een eerder opgelost code review patroon dat bij toekomstige reviews als context wordt meegenomen.

## Hoe dit werkt

1. `/review` vindt een issue en rapporteert het
2. De developer lost het issue op
3. `/compound` legt het patroon vast als solution document
4. Bij de volgende `/review` wordt het patroon meegenomen als context
5. Agents herkennen vergelijkbare patronen sneller en nauwkeuriger
